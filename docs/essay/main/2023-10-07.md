# 2023-10-07

## 在 Firefox 中使用原生的 Qt 文件选择器

首先确认是否安装了 `xdg-desktop-portal-kde`，然后在 KDE 编辑应用程序对话框的 **应用程序** -> **环境变量** 中，填入 `GTK_USE_PORTAL=1`。然后重启浏览器即可。
